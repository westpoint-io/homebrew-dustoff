# homebrew-dustoff

Homebrew formula for [dustoff](https://github.com/westpoint-io/dustoff) — find and remove JS/TS build artifacts wasting disk space.

## Install

```bash
brew install westpoint-io/dustoff/dustoff
```

Or:

```bash
brew tap westpoint-io/dustoff
brew install dustoff
```

## Uninstall

```bash
brew uninstall dustoff
brew untap westpoint-io/dustoff
```
